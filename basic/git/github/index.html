Github

Github setup

  1. Signup account with Github and create a repo “tool”

  2. Enable ssh connection to Github connect to Github through ssh 1

     ```
 # generate ssh keys
 cd ~/.ssh
 ssh-keygen -t rsa -C "wedgevine@outlook.com" -f "id_rsa_wedgevine"
 ```

  3. Adding a new SSH key to your GitHub account

  4. Per 1, create ~/.ssh/config file for each Github account

  5. Setting the git remote Url for the local repositories

     ```
 cd local-repo
 git init
 git config user.name "wedgevine"
 git config user.email "wedgevine@outlook.com"
 git remote add origin git@wedgevine.github.com:wedgevine/tool.git
 # note the above hostname notation which shoud match what is defined in config file
 ```

  6. Add key to ssh-agent, so don't have to input passphrase every time

     ```
 # start ssh agent
 eval "$(ssh-agent -s)"
 # check ssh keys added
 ssh-add -l
 # add ssh key
 ssh-add ~/.ssh/id_rsa_wedgevine
 ```

  7. Since the key has been stored in ssh-agent, we can do normal Git add/commit/push
